---
title: 'kline'
description: 'Subscribe to the latest klines (OHLC) updates of the subscribed symbol.'
---
Please note that for symbols with low trading activity, the Kline might be generated through the mid price of the orderbook in addition to any executed trades on the platform.


### Push interval: 

### Topic: kline@{symbol}@{time} e.g., kline@SPOT_BTC_USDT@1m
#### {time}: 1m/3m/5m/15m/30m/1h/2h/4h/6h/12h/1d/3d/1w/1M


<RequestExample>

```bash
{
    "topic": "kline@SPOT_BTC_USDT@1m",
    "ts": 1618822432146,
    "data":{
        "s": "SPOT_BTC_USDT",  // symbol
        "t": "1m",             // kline type
        "o": "56948.97",       // open
        "c": "56891.76",       // close
        "h": "56948.97",       // high
        "l": "56889.06",       // low
        "v": "44.00947568",    // volume in base token
        "a": "2504584.9",      // amount in USDT
        "st": 1618822380000,   // kline start timestamp
        "et": 1618822440000,   // kline end timestamp
        "ts": 1614152260000    // kline generation time
        "tts": 1614152250000   // last trade time
    }
}
```

</RequestExample>
